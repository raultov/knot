## Overview & Rationale

This plan adapts the standard Phase 2 (HTML Support) from `docs/specs/multilanguage_roadmap.md` to focus on **modern web frameworks** rather than pure HTML. The key insight is:

> For frameworks like Angular 2+, React, Next.js, and React Native, HTML rarely exists in isolation. Angular uses `.html` template files tightly coupled to TypeScript components, while React/Next.js use JSX/TSX directly. Therefore, our "HTML support" must be **dual-pronged**:
>
> 1. Parse `.html` files (for Angular templates and Web Components)
> 2. Extract DOM attributes (`id`, `className`, `class`) from JSX/TSX files (for React/Next.js)

This enables the AI to answer questions like: *"What components use this CSS class?"* across both Angular and React ecosystems.

---

## Analysis of Current Implementation

### Existing JSX/TSX Support
- ✅ JSX component invocations are already captured (`<ChartToolbar />`, `<Sheet.Content />`)
- ✅ React component calls create `CALLS` relationships in Neo4j
- ❌ **Missing**: HTML attributes like `id` and `className` are not extracted as entities
- ❌ **Missing**: No `.html` file support for Angular templates

### Target: Add HTML Attribute Entities
When parsing JSX/TSX, we need to:
- Extract `id="..."` attributes → Create `HtmlId` entities
- Extract `className="..."` or `class="..."` attributes → Create `HtmlClass` entities (split by whitespace)

When parsing `.html` files (Angular, Web Components), we need to:
- Extract custom elements (e.g., `<app-user-profile>`, `<web-component>`) → Create `HtmlElement` entities
- Extract `id="..."` attributes → Create `HtmlId` entities
- Extract `class="..."` attributes → Create `HtmlClass` entities (split by whitespace)

### 3. Create HTML Query File

**File**: `queries/html.scm`

```scm
; ============================================================
; HTML Tree-sitter extraction queries for Angular & Web Components
;
; Captures:
; - Custom elements (Web Components, Angular components)
; - id attributes
; - class attributes
;
; Standard HTML5 elements (div, span, p) are ignored to avoid
; saturating the knowledge graph with low-value entities.
; ============================================================

; Web Components / Custom Elements
; Matches: <app-user-profile>, <web-component>, <sheet-dialog>
; Ignores standard HTML tags (no hyphens)
((element
   tag_name: (tag_name) @html_element_name)
 (#match? @html_element_name "-"))

; id attributes on any element
(attribute
  name: (attribute_name) @attr_name
  value: (attribute_value) @html_id_value)
(#eq? @attr_name "id")

; class attributes on any element
(attribute
  name: (attribute_name) @attr_name
  value: (attribute_value) @html_class_value)
(#eq? @attr_name "class")

; Comments (for documentation)
(comment) @doc
```

### 4. Extract HTML Attributes from JSX/TSX

**File**: `src/pipeline/parser/languages/javascript.rs` and `typescript.rs`

Modify JSX element processing to capture attributes:

```rust
/// Extract HTML attributes (id, className) from JSX elements
/// 
/// When processing <Component id="main" className="flex-col">, we create:
/// - HtmlId entity for "main"
/// - HtmlClass entity for each class in "flex-col" (split by space)
pub(crate) fn extract_jsx_attributes(
    element_node: Node<'_>,
    source: &[u8],
    entities: &mut Vec<ParsedEntity>,
    file_path: &str,
    repo_name: &str,
) {
    let line = element_node.start_position().row + 1;
    
    // Find all jsx_attribute children
    let mut child = element_node.child(0);
    while let Some(c) = child {
        if c.kind() == "jsx_attribute" {
            if let Some(attr_name_node) = c.child_by_field_name("name") {
                let attr_name = node_text(attr_name_node, source);
                
                // Extract id attribute
                if attr_name == "id" {
                    if let Some(value_node) = c.child_by_field_name("value") {
                        let id_value = extract_string_value(value_node, source);
                        if !id_value.is_empty() {
                            let entity = ParsedEntity {
                                uuid: Uuid::new_v4(),
                                name: id_value.clone(),
                                kind: EntityKind::HtmlId,
                                fqn: format!("#{}", id_value),
                                signature: None,
                                docstring: None,
                                inline_comments: Vec::new(),
                                decorators: Vec::new(),
                                language: "jsx".to_string(),
                                file_path: file_path.to_string(),
                                start_line: line,
                                enclosing_class: None,
                                repo_name: repo_name.to_string(),
                                reference_intents: Vec::new(),
                            };
                            entities.push(entity);
                        }
                    }
                }
                
                // Extract className attribute (React convention)
                if attr_name == "className" || attr_name == "class" {
                    if let Some(value_node) = c.child_by_field_name("value") {
                        let class_str = extract_string_value(value_node, source);
                        // Split by whitespace and create entity for each class
                        for class_name in class_str.split_whitespace() {
                            if !class_name.is_empty() {
                                let entity = ParsedEntity {
                                    uuid: Uuid::new_v4(),
                                    name: class_name.to_string(),
                                    kind: EntityKind::HtmlClass,
                                    fqn: format!(".{}", class_name),
                                    signature: None,
                                    docstring: None,
                                    inline_comments: Vec::new(),
                                    decorators: Vec::new(),
                                    language: "jsx".to_string(),
                                    file_path: file_path.to_string(),
                                    start_line: line,
                                    enclosing_class: None,
                                    repo_name: repo_name.to_string(),
                                    reference_intents: Vec::new(),
                                };
                                entities.push(entity);
                            }
                        }
                    }
                }
            }
        }
        child = c.next_sibling();
    }
}
```

### 5. Create HTML Parser Module

**File**: `src/pipeline/parser/languages/html.rs` (NEW)

```rust
//! HTML/Angular template extraction for custom elements and attributes.

use tree_sitter::Node;
use crate::models::entity::{EntityKind, ParsedEntity};
use crate::utils::node_text;
use uuid::Uuid;

/// Extract entities from HTML files (Angular templates, Web Components)
pub(crate) fn extract_entities_html(
    root: Node<'_>,
    source: &[u8],
    file_path: &str,
    repo_name: &str,
) -> Vec<ParsedEntity> {
    let mut entities = Vec::new();
    extract_html_elements(root, source, &mut entities, file_path, repo_name);
    entities
}

/// Recursively extract custom elements and attributes from HTML AST
fn extract_html_elements(
    node: Node<'_>,
    source: &[u8],
    entities: &mut Vec<ParsedEntity>,
    file_path: &str,
    repo_name: &str,
) {
    if node.kind() == "element" {
        let line = node.start_position().row + 1;
        
        // Extract tag name (custom elements only - those with hyphens)
        if let Some(tag_node) = node.child_by_field_name("tag_name") {
            let tag_name = node_text(tag_node, source);
            if tag_name.contains('-') {
                // Web Component / Angular Component
                entities.push(ParsedEntity {
                    uuid: Uuid::new_v4(),
                    name: tag_name.clone(),
                    kind: EntityKind::HtmlElement,
                    fqn: format!("<{}>", tag_name),
                    signature: None,
                    docstring: None,
                    inline_comments: Vec::new(),
                    decorators: Vec::new(),
                    language: "html".to_string(),
                    file_path: file_path.to_string(),
                    start_line: line,
                    enclosing_class: None,
                    repo_name: repo_name.to_string(),
                    reference_intents: Vec::new(),
                });
            }
        }
        
        // Extract attributes (id, class)
        let mut child = node.child(0);
        while let Some(c) = child {
            if c.kind() == "attribute" {
                extract_html_attribute(c, source, entities, line, file_path, repo_name);
            }
            child = c.next_sibling();
        }
    }
    
    // Recurse to children
    let mut child = node.child(0);
    while let Some(c) = child {
        extract_html_elements(c, source, entities, file_path, repo_name);
        child = c.next_sibling();
    }
}

/// Extract id and class attributes from HTML elements
fn extract_html_attribute(
    attr_node: Node<'_>,
    source: &[u8],
    entities: &mut Vec<ParsedEntity>,
    line: usize,
    file_path: &str,
    repo_name: &str,
) {
    if let Some(name_node) = attr_node.child_by_field_name("name") {
        let attr_name = node_text(name_node, source);
        
        if attr_name == "id" {
            if let Some(value_node) = attr_node.child_by_field_name("value") {
                let id_value = extract_string_value(value_node, source);
                if !id_value.is_empty() {
                    entities.push(ParsedEntity {
                        uuid: Uuid::new_v4(),
                        name: id_value.clone(),
                        kind: EntityKind::HtmlId,
                        fqn: format!("#{}", id_value),
                        signature: None,
                        docstring: None,
                        inline_comments: Vec::new(),
                        decorators: Vec::new(),
                        language: "html".to_string(),
                        file_path: file_path.to_string(),
                        start_line: line,
                        enclosing_class: None,
                        repo_name: repo_name.to_string(),
                        reference_intents: Vec::new(),
                    });
                }
            }
        } else if attr_name == "class" {
            if let Some(value_node) = attr_node.child_by_field_name("value") {
                let class_str = extract_string_value(value_node, source);
                for class_name in class_str.split_whitespace() {
                    if !class_name.is_empty() {
                        entities.push(ParsedEntity {
                            uuid: Uuid::new_v4(),
                            name: class_name.to_string(),
                            kind: EntityKind::HtmlClass,
                            fqn: format!(".{}", class_name),
                            signature: None,
                            docstring: None,
                            inline_comments: Vec::new(),
                            decorators: Vec::new(),
                            language: "html".to_string(),
                            file_path: file_path.to_string(),
                            start_line: line,
                            enclosing_class: None,
                            repo_name: repo_name.to_string(),
                            reference_intents: Vec::new(),
                        });
                    }
                }
            }
        }
    }
}

/// Extract string value from attribute value node
fn extract_string_value(value_node: Node<'_>, source: &[u8]) -> String {
    let text = node_text(value_node, source);
    // Remove quotes if present
    text.trim_matches(|c| c == '"' || c == '\'' || c == '{' || c == '}').to_string()
}
```
